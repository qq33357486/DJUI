# 更新日志

## [0.5.1] - 2026-07-03

### 优化
- 按钮默认音效未配置时仅首次弹出说明，后续改为顶部栏非打断提醒。

## [0.5.0] - 2026-07-03

### 新增
- 颜色与透明度拆分为独立字段，遵循 PS 习惯（0%=全透明 100%=可见）
- 图片控件支持独立的图片透明度调节

### 优化
- 属性面板 UX 重组：外观与文本拆分，锚点与拉伸默认展开
- 整体透明度显示改为百分比（0-100%），存储仍为 0-1 兼容 Runtime

### 修复
- 修复颜色选择器在 alpha=0 时选新颜色强制改为不透明的问题
- 修复有图片时调整背景透明度无效的问题（改为绑定图片透明度）

## [0.4.0] - 2026-07-03

### 新增
- 自适应容器（auto-size）布局：容器可按子控件边界自动调整宽高
- 更新公告功能：编辑器版本升级时自动弹出时间线式变更日志
- 数据边界防护体系：normalize 归一化层 + AppErrorBoundary 防白屏

### 修复
- 修复老项目加载白屏问题（节点缺少 children 字段导致渲染崩溃）
- 修复锚点拉伸轴拖拽时坐标换算错误

### 优化
- 重构拖拽系统：支持父子节点联动预览，拖拽体验更流畅
- 纯前端架构迁移完成，移除后端服务

## [0.3.0] - 2026-06-29

### 新增
- 纯前端架构：使用 File System Access API 直接读写工程目录
- 声音配置系统：支持按钮点击音效绑定
- 九宫格切片编辑器
- AGENTS.md 规范自动更新机制
- 脚本区同步功能

### 优化
- 锚点系统重构为 NGUI 风格（anchor.side 9-way）
- Runtime 同步改用前端 File System Access API

## [0.2.0] - 2026-06-15

### 新增
- Docker 容器化部署
- 在线实例 https://ui.duojie.games
- GHCR 镜像发布

## [0.1.0] - 2026-06-10

### 新增
- 初始版本：Web Editor + C# Runtime + 素材处理脚本
- 可视化 UI 页面编辑（拖拽、属性面板、画布预览）
- JSON 导出 + Runtime 控件树构建
